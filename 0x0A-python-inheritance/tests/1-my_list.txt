# 1-my_list.txt

=====================
The ``MyList`` module
=====================

This library is very simple, since it only has one class
calles ``MyList``.

First import ``MyList`` from the ``1-my_list`` module:

	>>> MyList = __import__('1-my_list').MyList
	>>> my_list = MyList()
	>>> my_list.print_sorted()
	[]
	>>> my_list.extend([1, 4, -2, 3, 5])

Now use it:

Numbers
=======

::

	>>> my_list.print_sorted()
	[-2, 1, 3, 4, 5]
	>>> print(my_list)
	[1, 4, -2, 3, 5]
